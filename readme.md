# Asada Engine

This is a 2D oriented WebGL engine.

### Features 
* Sprites
* Animated sprites
* Fonts
* Lights (Spot, Directional, Point)
* Tiled maps
* Layer system
* Audio support
* Touch and gamepads support
* Cameras
* Scenes
* Particle system
* 2D Physics (Matterjs)
* File loader
